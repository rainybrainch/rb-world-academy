## Overview

This expanded Chinese curriculum provides comprehensive, authentic learning content that goes beyond basic greetings. Each lecture includes:

### Content Per Lecture
- **Vocabulary:** 8-12 items with:
  - Pinyin (with tone marks)
  - Simplified + Traditional characters
  - Synonyms and antonyms
  - Usage examples
  - Cultural context notes
  
- **Example Sentences:** 5-8 sentences with:
  - Chinese text
  - Pinyin
  - English translation
  - Natural usage context

- **Practice Exercises:** 3-Tier System
  - **Basic:** Foundational comprehension (for A1-A2)
  - **Intermediate:** Applied usage patterns (for A2-B1)
  - **Advanced:** Nuanced expression & mastery (for B1-B2)

- **Cultural Content:**
  - Regional variations
  - Idioms (成语 / chéngyǔ)
  - Traditional customs
  - Modern variations (internet slang, social media)

## Special Features

### 1. Character Variations System
Each vocabulary item includes:
```
traditional: '你好' (Traditional Chinese)
simplified: '你好' (Simplified Chinese)
```

This allows students to recognize both systems and understand the mapping.

### 2. Tone Mark System
All vocabulary includes:
```
toneMarks: 'nǐ = 3rd tone, hǎo = 3rd tone'
pinyin: 'nǐ hǎo'
```

### 3. Idiom Progression
- **A1 (Lesson 29):** 成语 Introduction - 10 basic idioms
- **A2 (Lesson 58):** 歇后语 Introduction - Puns and wordplay
- **B1 (Lesson 77):** 成语 Mastery - 200+ four-character idioms
- **B1 (Lesson 89):** 谚语 - Traditional Chinese proverbs
- **B2 (Lecture 109):** 成语 Usage - Advanced literary application

### 4. Cultural Context Integration
- **A1:** Basic etiquette and daily customs
- **A2:** Holidays (春节, Mid-Autumn Festival), dining customs
- **B1:** Historical events, martial arts, Fengshui, regional variations
- **B2:** Literature (Tang poetry, modern fiction), opera, calligraphy

### 5. Modern Language Variations
- **A2 (Lesson 54):** Internet & Social Media Language
- **B1 (Lesson 88):** Internet Slang & Millennial Language (网络用语)
- **B2 (Lesson 112):** Avant-Garde & Experimental Modern Chinese

### 6. Three-Tier Practice System

**Basic Level (Foundational Comprehension)**
- Vocabulary recognition
- Simple sentence translation
- Basic dialogue completion
- Character writing practice

Example:
```
"Practice saying greetings at different times of day"
Answer: "Greet 5 people using different time-appropriate expressions and responses."
```

**Intermediate Level (Applied Usage)**
- Combining multiple concepts
- Creating dialogues
- Short paragraph writing
- Analysis and comparison

Example:
```
"Learn to use appropriate measure words with different nouns"
Answer: "Create 10 sentences using numbers 1-10, each with a different noun 
and appropriate measure word. Include: people, animals, objects, and abstract items."
```

**Advanced Level (Mastery & Nuance)**
- Complex narrative writing
- Cultural analysis
- Metaphorical and idiomatic usage
- Comparative linguistics

Example:
```
"Explore abstract and emotional uses of color terms"
Answer: "Explain how colors carry emotional or metaphorical meanings in Chinese 
(e.g., 红色 for joy, 蓝色 for sadness). Create poetic sentences using colors metaphorically."
```
